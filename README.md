## 🧠 Questions & Answers

### 1️⃣ What is the difference between var, let, and const?
In JavaScript, var, let, and const are used to declare variables, but they behave differently.
- var is the old way of declaring variables. It is function-scoped and can be re-declared and updated. This can sometimes create bugs.
- let is block-scoped and more modern. It can be updated but cannot be declared again in the same scope.
- const is also block-scoped, but its value cannot be changed after assignment.

---

### 2️⃣ What is the spread operator (...)?
The spread operator (...) is used to expand elements of an array or object.
It helps to copy or combine data easily and makes the code shorter.
Example: If we have an array [1, 2, 3], we can create a new array like [1, 2, 3, 4] using spread.

---

### 3️⃣ What is the difference between map(), filter(), and forEach()?
All three are array methods, but they are used for different purposes.
- map() creates a new array by modifying each element  
- filter() creates a new array by selecting elements based on a condition  
- forEach() just loops through the array but does not return anything  

---

### 4️⃣ What is an arrow function?
An arrow function is a shorter way to write a function using =>. Instead of writing a full function, we can write it in one line.It makes the code cleaner and easier to read.

---

### 5️⃣ What are template literals?
Template literals are strings written using backticks (` `).
They allow us to insert variables inside a string using ${}.
They are useful for:
- writing dynamic strings  
- multi-line text  
- making code more readable
